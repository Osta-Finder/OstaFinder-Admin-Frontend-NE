# 🚀 تشغيل المشروع

## الخطوات السريعة

### 1️⃣ تثبيت المكتبات
```bash
npm install
```

### 2️⃣ تشغيل الخادم
```bash
npm run dev
```

### 3️⃣ فتح المتصفح
الداشبورد سيفتح تلقائياً على:
```
http://localhost:3000
```

## ✅ ما يجب أن تراه

عند فتح المشروع، يجب أن ترى:

1. **الشريط الجانبي (Sidebar)** على اليسار
   - لوحة القيادة
   - اعتمادات الفنيين
   - إدارة المستخدمين
   - التحليلات
   - إدارة الطلبات
   - الدعم الفني

2. **رأس الصفحة (Header)** في الأعلى
   - شريط البحث
   - الإشعارات
   - صورة المستخدم

3. **محتوى الصفحة (Main Content)**
   - بطاقات الإحصائيات
   - رسوم بيانية
   - جداول البيانات

## 🧪 اختبار الصفحات

### اختبر كل صفحة:

1. **لوحة القيادة** ✅
   - انقر على "لوحة القيادة" في الـ sidebar
   - يجب أن ترى بطاقات الإحصائيات والرسوم البيانية

2. **اعتمادات الفنيين** ✅
   - انقر على "اعتمادات الفنيين"
   - يجب أن ترى جدول الفنيين

3. **إدارة المستخدمين** ✅
   - انقر على "إدارة المستخدمين"
   - يجب أن ترى جدول المستخدمين

4. **التحليلات** ✅
   - انقر على "التحليلات"
   - يجب أن ترى رسوم بيانية

5. **إدارة الطلبات** ✅
   - انقر على "إدارة الطلبات"
   - يجب أن ترى جدول الطلبات

6. **الدعم الفني** ✅
   - انقر على "الدعم الفني"
   - يجب أن ترى نظام التذاكر

## 🔧 استكشاف الأخطاء

### إذا لم يعمل المشروع:

#### ❌ خطأ: "Cannot find module"
```bash
# الحل:
rm -rf node_modules
npm install
npm run dev
```

#### ❌ خطأ: "Port 3000 is already in use"
```bash
# الحل: استخدم منفذ مختلف
npm run dev -- --port 3001
```

#### ❌ خطأ: "Heroicons v2"
✅ تم حل هذا الخطأ بالفعل!

#### ❌ صفحات بيضاء
✅ تم حل هذا الخطأ بالفعل!
