# 🚀 البدء السريع

## المشكلة التي تم حلها
الصفحات الأخرى كانت تظهر بيضاء عند الضغط عليها. تم حل المشكلة بإضافة جميع الـ Redux slices إلى الـ store.

## الخطوات

### 1. تثبيت المكتبات
```bash
npm install
```

### 2. تشغيل الخادم
```bash
npm run dev
```

### 3. فتح المتصفح
الداشبورد سيفتح تلقائياً على `http://localhost:3000`

## الصفحات المتاحة

| الصفحة | المسار | الوصف |
|--------|--------|-------|
| لوحة القيادة | `/dashboard` | عرض الإحصائيات والرسوم البيانية |
| اعتمادات الفنيين | `/technicians` | إدارة طلبات اعتماد الفنيين |
| إدارة المستخدمين | `/users` | إدارة حسابات المستخدمين |
| التحليلات | `/analytics` | عرض التحليلات والرسوم البيانية |
| إدارة الطلبات | `/orders` | إدارة الطلبات والخدمات |
| الدعم الفني | `/support` | نظام الدعم الفني والتذاكر |

## الميزات

✅ **واجهة عربية RTL** - دعم كامل للغة العربية  
✅ **Responsive Design** - يعمل على جميع الأجهزة  
✅ **Redux State Management** - إدارة الحالة المركزية  
✅ **TailwindCSS** - تصميم حديث وجميل  
✅ **Headless UI** - مكونات قابلة للوصول  
✅ **React Router** - التنقل بين الصفحات  
✅ **Toast Notifications** - إشعارات المستخدم  
✅ **Recharts** - رسوم بيانية متقدمة  

## البيانات الوهمية

جميع الصفحات تحتوي على بيانات وهمية للاختبار:
- الطلبات: 3 طلبات
- الفنيين: 3 فنيين
- المستخدمين: 3 مستخدمين
- الرسوم البيانية: بيانات أسبوعية

## الأوامر المتاحة

```bash
# تشغيل الخادم
npm run dev

# بناء المشروع
npm run build

# معاينة البناء
npm run preview

# فحص الأخطاء
npm run lint
```

## الملفات المهمة

- `src/App.jsx` - التطبيق الرئيسي
- `src/store/store.js` - إعدادات Redux
- `src/components/Layout/MainLayout.jsx` - التخطيط الرئيسي
- `src/pages/` - جميع الصفحات
- `tailwind.config.js` - إعدادات Tailwind

## الدعم

إذا واجهت أي مشاكل:
1. تأكد من تثبيت المكتبات: `npm install`
2. امسح ذاكرة التخزين المؤقت: `rm -rf node_modules && npm install`
3. أعد تشغيل الخادم: `npm run dev`
